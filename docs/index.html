options.cache Object An object with cachable functions or a Redis cache of some sorts

getLeaguesBySummonerId¶

getLeaguesBySummonerId(region, summonerIds) ⇒ Promise Returns all leagues for specified summoners and summoners’ teams. Entries for each requested participant (i.e., each summoner and related teams) will be included in the returned leagues data, whether or not the participant is inactive. However, no entries for other inactive summoners or teams in the leagues will be included.

Kind: instance method of Ashe
Returns: Promise - A Promise
Fulfil: Object
Reject: Error
See: Riot Api: https://developer.riotgames.com/api/methods#!/985/3351

Param Type Description
region string The region to search in
summonerIds Array.<String> A list of summoner ids to get data for

getMatchList¶

getMatchList(region, summonerId, options) ⇒ Promise Gets the list of matches for a summoner in a region

NOT COMPLETE YET

A number of optional parameters are provided for filtering. It is up to the caller to ensure that the combination of filter parameters provided is valid for the requested summoner, otherwise, no matches may be returned. If either of the beginTime or endTime parameters is set, they must both be set, although there is no maximum limit on their range. If the beginTime parameter is specified on its own, endTime is assumed to be the current time. If the endTime parameter is specified on its own, beginTime is assumed to be the start of the summoner’s match history.

Kind: instance method of Ashe
Returns: Promise - A Promise
Fulfil: Object
Reject: Error
See: Riot API: https://developer.riotgames.com/api/methods#!/1069/3683

Param Type Description
region String the region to search in
summonerId String the summoner id to request data for

getSummonersByName¶

getSummonersByName(region, summonerNames) ⇒ Promise Get summoner objects mapped by standardized summoner name for a given list of summoner names

NOTE: The function already handles standardizing the names. Returned data will have the same key as the name provided

The response object contains the summoner objects mapped by the standardized summoner name, which is the summoner name in all lower case and with spaces removed. Use this version of the name when checking if the returned object contains the data for a given summoner. This API will also accept standardized summoner names as valid parameters, although they are not required.

Kind: instance method of Ashe
Returns: Promise - A Promise
Fulfil: Object
Reject: Error
See: Riot API: https://developer.riotgames.com/api/methods#!/1079/3722

Param Type Description
region String The region to search in
summonerNames Array.<String> A list of summoner names
